# itsdezen/homebrew-tap

Homebrew tap for [tili](https://github.com/itsdezen/tili), an i3-like
tiling window manager for macOS.

```sh
brew install itsdezen/tap/tili
```

See tili's [CONTRIBUTING.md](https://github.com/itsdezen/tili/blob/main/CONTRIBUTING.md#release-engineering)
for how this formula gets updated after each release.
